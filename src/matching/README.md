# 字符串问题 - 字符串匹配算法

## 问题描述

- 输入：文本字符串 `s1`, 模式字符串 `s2`.
- 输出：若 `s2` 为 `s1` 的子串，返回其首次出现的下标；否则返回 None.

## 算法

字符串匹配问题有很多算法：

- [朴素枚举](./simple.md)
- [Rabin-Karp](./rk.md)
- 有限状态自动机
- [Kunth-Morris-Pratt](./kmp.md)
