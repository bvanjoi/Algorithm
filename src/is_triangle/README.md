# 几何 - 可否为三角形

## 算法

- 输入：三个 i32 类型的变量 `a`, `b`, `c`.
- 输出：判断三者可否组成三角形，若可以，返回 `true`, 否则返回 `false`.

依据三角形的特征：

- 两边之和大于第三边；
- 两边之差小于第三边；
- 三条边均为正数。

写成 `a > 0 && b > 0 && c > 0 && a + b > c && (a - b).abs() < c`.

## 示例

当输入为 `2,3,4` 时， 满足 2 + 3 = 5 > 4,  (2 - 3).abs() = -1 < 4, 因此可以组成三角形。
