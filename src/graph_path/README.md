# 搜索问题 - 有向无环图的路径

## 有向无环图

在图论中：

- 有向图：每条边带有一条方向；
- 无环图：从某一顶点出发，无法经过若干条边再回到该点。

> 默认为连通有向无环图。

## 有向无环图的路径

- 输入：一个二维数组 `graph`, 下标 i 表示节点 i, 元素 `graph[i]` 表示节点 i 能到达的其他节点；输入进行了简化处理，节点 0 为起点，节点 `graph.len() - 1` 为终点。
- 输出：节点 0 到终点的所有路径。

### 算法

采用深度优先搜索算法，搜索所有可以达到终点的路径。

由于无环，所以不需要考虑是否走过。

### 示例

例如：

```txt
    0
  /   \
 1     2
  \   /
    3
```

上述示例中，从 0 点出发，到达 3 的所有路径。
